Weather Station Comparison App

Student: Alan O'Connell  |  ID: R00243626  |  Module: Interface Software Development L8  |  Variant: 16

Brief

A WPF C# application to compare weather data from Sherkin Island and Roches Point stations for the last 12 hours (temperature, wind speed, rainfall).

Data sources:

  • Met Eireann - Sherkin Island
  • Met Eireann - Roches Point

Specification

FeatureMarks
Load Data from SherkinWeatherData/ and RochesWeatherData/ folders into ListBoxes14
Double-click item in ListBox to select parameter for extraction12
Extract time + values, build SortedDictionary key/value pairs14
Display parameter name in label4
Plot dual-series chart using LiveCharts v0.9.714
Slider (potentiometer) to set Count between 1 and max elements16
Range Chart showing first N readings10
Write difference values to file using StreamWriter6
Menu: Load Stations, Plot Stations, Exit Application8
